# CSS Only Section Toggle

This is a beginner-friendly project that demonstrates how to show and hide different sections of a webpage using **only HTML and CSS**, without JavaScript.

When a particular div is clicked, a specific section is displayed while others remain hidden.  
This behavior is similar to basic DOM manipulation, but it is achieved using CSS properties like `display`, `:target`, or `:checked`.

---

##  Features

- No JavaScript used
- Click-based section switching
- Uses pure HTML and CSS
- Beginner-friendly project
- Responsive layout (optional)

---

##  Technologies Used

- HTML5  
- CSS3  

---

##  How It Works

- Each clickable div is linked to a section
- CSS controls which section is visible
- Sections are shown or hidden using:
  - `display: none`
  - `display: block`
  - or CSS pseudo-classes like `:target` / `:checked`
